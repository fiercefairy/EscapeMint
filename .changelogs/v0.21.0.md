# Release v0.21.0 - Intro Wizard, SPY Support & Backtest Presets

Released: 2026-01-15

## Overview

This release adds an interactive onboarding wizard with animated D3 visualizations, SPY (S&P 500 ETF) support, and preset configurations for the backtest tool.

## New Features

### Interactive Intro Wizard
- **12-step onboarding guide**: Explains the DCA strategy from basics to advanced concepts
- **Animated D3 charts**: Visual demonstrations of market growth, volatility, and strategy modes
  - MarketGrowthChart: Animated line showing historical market growth
  - VolatilityComparison: Straight vs volatile growth comparison
  - BuySellZones: Buy dips/sell peaks visualization
  - LeverageComparison: Real SPY vs SPXL data comparison
  - ModeComparison: Accumulate vs Harvest mode using actual backtest engine
- **URL-based navigation**: Each step has a shareable URL (`/intro/1`, `/intro/5`, etc.)
- **First-visit auto-show**: New visitors are guided to the intro before the backtest
- **"Learn How It Works" button**: Always accessible in the backtest header
- **Prominent disclaimer**: Not investment advice warnings on first step, last step, and footer
- **Fixed bottom navigation**: Navigation buttons stay fixed at bottom while content scrolls

### SPY Support
- **S&P 500 tracking**: Added SPY (non-leveraged S&P 500 ETF) as a backtest asset
- **Historical data**: 261 weekly data points with dividend reinvestment
- **SPY preset**: One-click preset for 100% SPY allocation with 15% target APY (harvest) or 10% (accumulate)
- **Allocation slider**: New SPY slider in PieBuilder component

### Backtest Presets
- **Quick configuration presets**: 5 preset buttons (TQQQ, SPXL, SPY, BTC, Blend) for each mode
- **Mode-aware settings**: Each preset automatically adjusts values based on Accumulate or Harvest mode

### Active Preset Indicator
- **Visual feedback**: Active preset button is highlighted with mode color (blue for Accumulate, orange for Harvest)
- **Smart deactivation**: Active state automatically clears when user manually changes any preset-controlled value

### Date Range Presets
- **Extended range options**: Added 3Y and 4Y buttons to the date range picker
- Range presets now include: YTD, 1Y, 2Y, 3Y, 4Y, ALL

### Mobile Responsive Improvements
- **Responsive header**: "Learn How It Works" link moves below tagline on mobile
- **Chart legends**: Wrap and stack vertically on small screens
- **Intro text sizing**: Smaller fonts and padding on mobile devices
- **Backtest iframe margins**: Fixed horizontal overflow on mobile

## Bug Fixes

- **ModeComparison charts**: Fixed $0K values after adding SPY (missing spyPct property)
- **Cursor pointer**: Added missing cursor pointer to date range preset buttons

## Technical Details

### New Files
- `pages/src/BacktestApp.tsx` - Extracted backtest functionality from App.tsx
