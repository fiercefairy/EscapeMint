# Release v0.21.0 - Backtest Presets

Released: 2026-01-15

## Overview

This release adds preset configurations to the backtest tool, making it easy to quickly switch between optimized settings for different assets and strategies.

## New Features

### Backtest Presets
- **Quick configuration presets**: Replace the single "Reset to Defaults" button with 4 preset buttons for each mode
- **Mode-aware settings**: Each preset automatically adjusts values based on whether you're in Accumulate or Harvest mode

#### Harvest Mode Presets
| Preset | Allocation | Target APY | DCA Tiers (Min/Mid/Max) |
|--------|------------|------------|-------------------------|
| TQQQ   | 100% TQQQ  | 52%        | $100/$100/$350          |
| SPXL   | 100% SPXL  | 44%        | $100/$100/$200          |
| BTC    | 100% BTC   | 80%        | $100/$100/$200          |
| Blend  | Mixed      | 40%        | $100/$200/$250          |

#### Accumulate Mode Presets
| Preset | Allocation | Target APY | DCA Tiers (Min/Mid/Max) |
|--------|------------|------------|-------------------------|
| TQQQ   | 100% TQQQ  | 20%        | $100/$100/$100          |
| SPXL   | 100% SPXL  | 20%        | $100/$100/$100          |
| BTC    | 100% BTC   | 30%        | $100/$100/$100          |
| Blend  | Mixed      | 20%        | $100/$200/$250          |

### Active Preset Indicator
- **Visual feedback**: Active preset button is highlighted with mode color (blue for Accumulate, orange for Harvest)
- **Smart deactivation**: Active state automatically clears when user manually changes any preset-controlled value
- Compares allocation percentages, target APY, and all three DCA tier values
