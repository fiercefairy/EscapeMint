# Release v0.40.x

### Changed

- **Remove automatic version bumping from CI**: Version numbers in package.json are now managed manually via dev commits instead of being auto-incremented by GitHub Actions on every push to dev or after release
- **CI permissions downgraded**: CI workflow no longer needs write access since it no longer pushes version bump commits
- **PLAN.md rewrite**: Condensed completed work into archive, removed sections now covered by GOALS.md (roadmap, brainstormed ideas, future features, tech stack), added concrete Next Actions section

### Added

- **GOALS.md**: Project goals document covering mission, core tenets, v1.0/v2.0 milestones, long-term vision, and non-goals

### Fixed

- **Replace pnpm workspace protocol**: Replaced pnpm `workspace:*` with npm-compatible `*` references, fixing `EUNSUPPORTEDPROTOCOL` error for npm users (#34)
- **Migrate CI to npm**: Replaced pnpm with npm in all CI workflows, removed `pnpm-lock.yaml`
- **Remove unused `jsonwebtoken` dependency** from server package
