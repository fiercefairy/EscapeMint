# Release v0.33.5

## v0.33.5

### Added

- **Auto-deposit checkbox in Take Action form**: When recording a BUY action for a fund that uses platform-managed cash, the form now detects if there's insufficient cash available. If so, it displays a checkbox option to automatically record a deposit to the platform's cash fund to cover the shortfall before recording the trade.

### Fixed

- **APY calculation consistency**: Entry list APY values now match Current Status widget for harvest mode funds that recycle capital across cycles
- **Coinbase import consolidation**: Same-day REBATE, FUNDING, and INTEREST entries are now consolidated into single entries per type per day
- **Dynamic backtest hostname**: Backtest page uses current hostname instead of hardcoded localhost
- **Server startup with tsx**: Server runs directly from TypeScript source using tsx, avoiding Node.js v25 compilation issues
- **Custom cash fund support**: Auto-deposit now correctly uses fund's configured `cash_fund` instead of always defaulting to platform-cash
