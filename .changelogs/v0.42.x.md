# Release v0.42.x

### Changed

- **Backtest page embedded inline**: The backtest page now renders directly within the app layout instead of using an iframe (dev) or redirecting to GitHub Pages (prod). All backtest engine code, components, and historical data are bundled into the web package.

### Fixed

- **Comma-formatted numbers in action form**: Values entered with commas (e.g., "1,000") are now parsed correctly instead of truncating at the comma
