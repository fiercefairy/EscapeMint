# Release v0.15.0 - Standalone Backtest Tool

Released: 2026-01-12

## Overview

This release introduces a standalone browser-based backtesting tool that allows users to test DCA strategies with historical data for SPXL, TQQQ, and BTC. The tool runs entirely in the browser and can be deployed to GitHub Pages.

## ✨ New Features

### Backtest Application (`pages/`)
- **Standalone SPA**: Self-contained React application with its own build system
- **GitHub Pages deployment**: Automated workflow deploys to `/EscapeMint/backtest/`
- **Historical data**: Bundled weekly price data for SPXL, TQQQ, and BTC (2021-2026)
- **Dividend support**: Includes historical dividend data for SPXL and TQQQ

### Interactive Configuration
- **Pie allocation builder**: Drag sliders to set SPXL/TQQQ/BTC percentages (must sum to 100%)
- **Strategy parameters**: Initial cash, target APY (0-100%), minimum profit threshold
- **DCA tiers**: Configure min/mid/max buy amounts based on drawdown levels
- **Sell mode toggle**: Switch between Accumulate (partial sells) and Harvest (full liquidation)
- **Cash APY**: Set interest rate earned on cash holdings
- **Date range picker**: Select backtest period with YTD, 1Y, 2Y, All presets

### Settings Persistence
- **localStorage**: All settings automatically saved and restored
- **Reset to defaults**: One-click button to restore default configuration

### Visualization
- **Value & Allocation chart**: Shows equity, invested amount, cash, and target line
- **Captured Profit chart**: Stacked area showing extracted profits, interest, and dividends
- **Gain Breakdown chart**: Multi-line chart showing Liquid, Unrealized, and Realized gains
- **APY Breakdown chart**: Multi-line chart showing Liquid, Unrealized, and Realized APY
- **Entries table**: Detailed weekly breakdown with all metrics

### Metrics Dashboard
- 8-column responsive grid showing key performance indicators
- Final Value, Liquid APY, Realized APY, Unrealized Gain
- Realized Gain, Liquid P&L, Total Invested, Total Extracted

## 🔧 Improvements

### CSS & Styling
- **Tailwind v4**: Uses new `@import "tailwindcss"` and `@source` directives
- **Custom slider styling**: Blue-bordered thumb with proper vertical centering
- **Legend indicators**: Explicit CSS ensures color squares render in production
- **Mobile responsive**: 2-column metric grid on mobile, 8-column on desktop

### Build System
- **Vite configuration**: Separate dev/prod builds with GitHub Pages base path
- **Engine integration**: Imports from `@escapemint/engine` via path alias
- **Shared components**: Reuses chart components from web package via `~web` alias
- **Cache busting**: Timestamp query params prevent stale data in browser

### Backtest Calculations

**Realized Gain**: `(totalExtracted - soldCostBasis) + cumCashInterest + cumDividends`
- Includes profit from sales, interest earned, and dividends received

**Liquid Gain**: `fundSize - initialCash`
- Total portfolio value minus starting capital

**APY Calculations**: `gain / initialCash / yearsElapsed`
- Annualized return based on elapsed time

**Dividend Tracking**:
- Converts buy amounts to equivalent SPXL/TQQQ shares
- Applies dividend payments based on ex-dates
- Dividends added to cash, tracked cumulatively

## 📦 Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
```

### Run Backtest Locally
```bash
cd pages
npm install
npm run dev
# Open http://localhost:5552
```

### Build for Production
```bash
cd pages
npm run build
# Output in pages/dist/
```
