# Release v0.22.0 - BRGNX Support & Leverage Chart Improvements

Released: 2026-01-15

## Overview

This release replaces SPY with BRGNX (Russell 1000) in the backtest tool for more accurate SPXL comparison, and adds a Price/DCA toggle to the leverage comparison chart.

## New Features

### BRGNX (Russell 1000) Support
- **Replaced SPY with BRGNX**: Since SPXL tracks the Russell 1000 index (not S&P 500), BRGNX provides a more accurate 1x baseline comparison
- **BRGNX preset**: New preset button with 10% target APY for both accumulate and harvest modes
- **Historical data**: 261 weekly data points for BRGNX added to backtest tool
- **BRGNX dividends**: Added 21 quarterly dividend payments (2021-2025) for accurate backtest calculations

### Leverage Comparison Chart Toggle
- **Price mode**: Shows normalized prices starting at 100 (original behavior)
- **DCA mode**: Simulates investing $100/week in each fund, showing equity value over time
- Toggle buttons above the chart to switch views instantly
- Shows total invested amount in DCA mode footer

### Official Fund Descriptions
- **TQQQ**: Added official fund objective description from ProShares
- **SPXL**: Added official fund objective description from Direxion, corrected to show Russell 1000 (not S&P 500)
- External comparison links to portfolioslab.com for both funds

## Bug Fixes

- **Config grid overlap**: Changed from lg:grid-cols-4 to xl:grid-cols-4 to prevent slider overlap on typical laptop screens
- **Slider label truncation**: Widened labels from 80px to 85px to fit "Min (≥ target)" etc.
- **BRGNX preset APY**: Fixed to use 10% for both modes (was 15% in harvest)
