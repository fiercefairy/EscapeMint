# Release v0.1.0 - Initial Release

Released: 2025-09-01

## Overview

The first official release of EscapeMint - a local-first, open-source capital allocation engine for rules-based fund management.

## 🎉 Features

### Core Calculation Engine
- **Pure Functions**: Zero-dependency calculation library with no side effects
- **DCA Strategy**: Tiered dollar-cost averaging based on performance
- **Target-Based Trading**: Automatic buy/sell recommendations based on target APY
- **Gain Tracking**: Real-time tracking of realized and unrealized gains

### Data Storage
- **TSV Files**: Human-readable tab-separated value format
- **File Locking**: Concurrent access protection with `proper-lockfile`
- **Local-First**: All data stays on your machine
- **Portable**: Plain text files you can backup, inspect, and version control

### API Server
- **Express Backend**: RESTful API for fund operations (port 5551)
- **CRUD Operations**: Create, read, update, and delete funds and entries
- **Calculation Endpoints**: Server-side computation of metrics and recommendations
- **Entry Management**: Add, edit, and delete fund entries

### Web Frontend
- **React UI**: Modern, responsive single-page application (port 5550)
- **Dashboard**: Overview of all funds with key metrics
- **Fund View**: Detailed fund page with entries, charts, and audit trail
- **Entry Forms**: Add and edit transactions with validation
- **Audit Trail**: Complete history of all fund activities

### Process Management
- **PM2 Integration**: Automatic restart on file changes
- **Development Workflow**: Simple commands for start/stop/restart
- **Log Management**: Centralized logging with PM2

## 📦 Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```

The app will be available at:
- Frontend: http://localhost:5550
- API: http://localhost:5551

## 🚀 Getting Started

1. Create your first fund from the dashboard
2. Set your DCA amounts and target APY
3. Add entries to track your investments
4. Follow the recommendations to stay on track
