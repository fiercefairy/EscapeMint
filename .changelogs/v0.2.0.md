# Release v0.2.0 - Storage Refactor & Cash Flow Tracking

Released: 2025-10-01

## Overview

Major storage architecture update separating data from configuration, plus new cash flow tracking capabilities and improved visualization.

## 🎉 New Features

### Storage Refactor
- **Separate TSV and JSON Files**: Data entries in `.tsv`, configuration in `.json`
- **Cleaner Data Model**: Better separation of concerns between time-series data and settings
- **Easier Editing**: Configuration can be modified without touching historical data
- **Better Performance**: Faster reads and writes with optimized file structure

### Cash Flow Tracking
- **DEPOSIT/WITHDRAW Actions**: Track cash movements in and out of funds
- **Accurate Capital Tracking**: Distinguish between invested capital and market gains
- **Historical Cash Flow**: View complete cash movement history

### UI Improvements
- **Column Reordering**: Drag-and-drop column reordering in entries table
- **Enhanced Charts**: New visualizations for value, allocation, and performance
- **Improved Audit Trail**: Better visibility into fund activity and changes

## 🔧 Improvements

- More maintainable codebase with clearer data/config separation
- Foundation for advanced cash management features
- Better user experience with customizable tables

git clone https://github.com/atomantic/EscapeMint.git
