# Release v0.19.0 - Chart & Table UX Improvements

Released: 2026-01-13

## Overview

This release focuses on UX improvements across charts and tables, making key metrics more visible at a glance and improving table navigation.

## 🎉 New Features

### Chart Legend Values
- **Value & Allocation chart**: Legend now shows latest Value, Invested, and Target amounts directly
- **Captured Profit chart**: Legend now shows latest Extracted, Interest, and Dividends amounts
- Quickly see current totals without hovering over the chart

### Backtest Entries Table
- **Sticky header**: Table header stays fixed when scrolling through entries
- **Sortable date column**: Click Date header to toggle ascending/descending order
- **Visual sort indicator**: Arrow icon (▲/▼) shows current sort direction

## 🔧 Improvements

### Fund Detail Page
- **Smart Cash display**: Cash stat hidden in Current State when fund doesn't manage cash (always $0)
- Cleaner stats grid for funds using external cash sources
- **Pre-filled Take Action form**: Form now auto-fills equity, cash, shares, and margin values from the latest entry for quick updates

### Coinbase Derivatives Scraping
- **Cash balance fetching**: After scraping transactions, automatically fetches current cash balance from Coinbase home page
- Cash balance displayed in scrape completion summary

### Documentation
- Clarified that derivatives/futures is an advanced feature currently only supported on Coinbase
- Documented that primary data import method is browser scraping from coinbase.com

### Deployment
- **GitHub Pages deployment now triggers on main branch**: More stable production releases

## 📦 Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```
