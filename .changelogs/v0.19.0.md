# Release v0.19.0 - Chart & Table UX Improvements

Released: 2026-01-13

## Overview

This release focuses on UX improvements across charts and tables, making key metrics more visible at a glance and improving table navigation. Also includes significant improvements to derivatives fund tracking with accurate cash balance from Coinbase.

## New Features

### Chart Legend Values
- **Value & Allocation chart**: Legend now shows latest Value, Invested, and Target amounts directly
- **Captured Profit chart**: Legend now shows latest Extracted, Interest, and Dividends amounts
- Quickly see current totals without hovering over the chart

### Backtest Entries Table
- **Sticky header**: Table header stays fixed when scrolling through entries
- **Sortable date column**: Click Date header to toggle ascending/descending order
- **Visual sort indicator**: Arrow icon (▲/▼) shows current sort direction

## Improvements

### Fund Detail Page
- **Smart Cash display**: Cash stat hidden in Current State when fund doesn't manage cash (always $0)
- Cleaner stats grid for funds using external cash sources
- **Pre-filled Take Action form**: Form now auto-fills equity, cash, shares, and margin values from the latest entry for quick updates

### Coinbase Derivatives Scraping
- **Cash balance fetching**: After scraping transactions, automatically navigates to coinbase.com/home to fetch current cash balance
- Creates a HOLD entry with updated cash balance if it differs from last known value
- Cash balance displayed in scrape completion summary

### Derivatives Fund Display
- **Accurate Cash display**: Shows tracked cash balance from Coinbase when available, falling back to calculated margin balance
- **Accurate Equity**: Uses tracked cash + unrealized P&L when cash is tracked
- **Accurate Available Funds**: Uses tracked cash - margin locked when cash is tracked
- **Accurate Liquidation Price**: Recalculates using tracked cash for more accurate values
- **Liquidation Price styling**: Negative values (over-collateralized) shown in green to indicate safety

### Documentation
- Clarified that derivatives/futures is an advanced feature currently only supported on Coinbase
- Documented that primary data import method is browser scraping from coinbase.com

### Deployment
- **GitHub Pages deployment now triggers on main branch**: More stable production releases

## Bug Fixes

### Coinbase Transaction Scraper
- **Fixed BUY/SELL dialog detection**: Scraper now correctly identifies trade dialogs by targeting the `h1` title element instead of generic headline classes that matched wrong elements
- **Fixed cash balance not saving**: Fund is now loaded whenever `fundId` is provided, ensuring cash balance updates are saved even when no new transactions are found
