# Release v0.19.0 - Backtest UX Improvements

Released: 2026-01-13

## Overview

This release improves the backtest page user experience with better table navigation and updates the deployment workflow to deploy from main branch.

## 🎉 New Features

### Backtest Entries Table
- **Sticky header**: Table header now stays fixed at the top when scrolling through entries, making it easier to reference column names
- **Sortable date column**: Click the Date header to toggle between ascending (oldest first) and descending (newest first) order
- **Visual sort indicator**: Arrow icon (▲/▼) shows current sort direction

## 🔧 Improvements

### Deployment
- **GitHub Pages deployment now triggers on main branch**: Changed from deploying on dev branch to main branch for more stable production releases

## 📦 Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```
