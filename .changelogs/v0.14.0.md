# Release v0.14.0 - M1 Test Platform & Margin Integrity Checks

Released: 2026-01-12

## Overview

This release adds a new m1test platform with margin-enabled stock fund simulation and introduces margin integrity warnings in the UI to highlight potential margin call situations.

## ✨ New Features

### M1 Test Platform
- **New m1test-pie fund**: Blended stock fund (50% BTC + 25% TQQQ + 25% SPXL) with margin support
- **New m1test-cash fund**: Cash account that provides margin access to the pie fund
- **Margin mechanics**:
  - 48% of pie equity available as margin
  - 25% buffer on margin utilization to prevent margin calls
  - 5% APR margin interest (accrues monthly)
  - 4% APY cash interest
  - Dividends and sell proceeds pay down margin first
- **Blended price history**: Created pie-weekly.json with normalized blended returns
- **Generation script**: `scripts/generate-m1test.cjs` for regenerating test data

### Margin Integrity Warnings
- **New data integrity check**: Highlights entries where margin borrowed exceeds margin available
- **Orange warning banner**: Distinct from red fund size warnings, shows count of affected entries
- **Row highlighting**: Orange background for margin call entries in the entries table
- **Tooltip**: Hover shows exact borrowed vs available amounts

## 🔧 Improvements

### Fund Configuration
- Updated m1test-pie.json with $20,000 fund size
- Margin buffer prevents new borrowing when utilization exceeds 75%
