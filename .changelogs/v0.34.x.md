# Release v0.34.x

## v0.34.0

### Added

- **Disclaimer in README and WelcomePanel**: Added clear disclaimer noting this app is shared publicly for friends and family, is not investment advice, and requires comfort with holding paper losses through downturns

## v0.34.1

### Added

- **Formula expressions in entry forms**: All numeric input fields accept `=` formulas (e.g., `=500.97+459.55`) with `+`, `-`, `*`, `/` and live computed result hint
- **Coinbase position scraping**: Scrape liquidation price, mark price, and unrealized P&L from Coinbase Advanced Trade positions table

### Fixed

- **Derivatives unrealized P&L**: Use current mark price instead of stale values; support both long and short positions
- **Derivatives maintenance margin**: Calculate based on absolute notional value for both long and short positions
- **Liquidation price**: Use server-scraped liquidation price instead of frontend recalculation
