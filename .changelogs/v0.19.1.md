# Release v0.19.1 - Derivatives Cash Balance Tracking

Released: 2026-01-14

## Overview

This release improves derivatives fund tracking by automatically fetching and using the actual cash balance from Coinbase, resulting in more accurate equity, liquidation price, and available funds calculations.

## Bug Fixes

### Coinbase Transaction Scraper
- **Fixed BUY/SELL dialog detection**: Scraper now correctly identifies trade dialogs by targeting the `h1` title element instead of generic headline classes
- **Fixed cash balance not saving**: Fund is now loaded whenever `fundId` is provided, ensuring cash balance updates are saved even when no new transactions are found

### Derivatives Fund Display
- **Fixed Cash column**: Now shows tracked cash balance from entry when available, falling back to calculated margin balance
- **Fixed Margin Balance in Current State**: Uses tracked cash instead of calculated value
- **Fixed Equity calculation**: Uses tracked cash + unrealized P&L when cash is tracked
- **Fixed Available Funds calculation**: Uses tracked cash - margin locked when cash is tracked
- **Fixed Liquidation Price calculation**: Recalculates using tracked cash for more accurate values

## Improvements

### Cash Balance Auto-Update
- After scraping Coinbase transactions, automatically navigates to coinbase.com/home to fetch current cash balance
- Creates a HOLD entry with the updated cash balance if it differs from the last known value
- Cash balance displayed in scrape completion summary

### Liquidation Price Display
- Negative liquidation prices (over-collateralized positions) now shown in green
- Helps users track their safety margin from $0 liquidation

## Technical Details

For derivatives funds, when `entry.cash` is available (from scraping), the following calculations now use it instead of the transaction-derived `marginBalance`:
- `derivEquity = entry.cash + unrealizedPnl`
- `derivAvailableFunds = entry.cash - marginLocked`
- `derivLiquidationPrice = avgEntry - (entry.cash - maintenanceMargin) / notionalSize`

This aligns EscapeMint's display with Coinbase's actual account values.
