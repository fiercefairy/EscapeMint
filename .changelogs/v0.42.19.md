# Release v0.42.19

Released: 2026-03-01

## Overview

Fixes React runtime crash caused by Dependabot bumping `react` to 19.x while `react-dom` stayed at 18.x, and aligns `@vitest/coverage-v8` with vitest 4.x.

## Bug Fixes

- **Fix React 19/ReactDOM 18 version mismatch**: Dependabot bumped `react` to `^19.2.4` without bumping `react-dom`, causing `TypeError: Cannot read properties of undefined (reading 'ReactCurrentBatchConfig')` at runtime. Pinned both back to `^18.3.1` along with `@types/react` to `^18.3.18`.
- **Align @vitest/coverage-v8 with vitest 4.x**: All four packages had `@vitest/coverage-v8@2.1.9` which conflicts with `vitest@^4.0.18`. Updated to `^4.0.18` across engine, server, storage, and web packages.

## Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```
