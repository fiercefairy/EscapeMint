# Release v0.13.0 - Dashboard APY Fix & M1 Pie Update

Released: 2026-01-11

## Overview

This release fixes an extreme APY calculation bug in the aggregate dashboard chart and updates the M1 pie share link.

## 🐛 Bug Fixes

### Dashboard APY Calculation
- **Fixed extreme APY values**: The aggregate "All Platforms" APY chart was showing values like -36500% on early data points
  - Root cause: Annualized APY calculation used `(365 / daysElapsed)` multiplier, which created extreme values when `daysElapsed` was small (e.g., 1-6 days)
  - Fix: Added `MIN_DAYS_FOR_APY = 7` threshold - APY is only calculated after 7 days have elapsed from the filter start date
  - This prevents misleading spikes in the first week of any filtered time range

### Data Correction
- Fixed `cryptocom-btc.tsv` initial entry which had `value=0` instead of `value=600` on the first BUY, contributing to the extreme APY calculation

### TypeScript Fix
- Fixed pre-existing TypeScript error in `DerivativesPriceChart.tsx` where `yDomainMin`/`yDomainMax` could be undefined
