# Release v0.42.21

Released: 2026-03-01

## Overview

Fixes fund size showing the wrong value in the dashboard table view and aggregate API for cash funds.

## Bug Fixes

- **Dashboard table shows actual fund size instead of config value**: The table view used `config.fund_size_usd` (static) instead of `latestFundSize` (actual). For cash funds like SAVE where fund size equals the cash balance, this showed a stale value that didn't match the fund card or fund detail page.
- **Aggregate API overrides fundSize from computeFundFinalMetrics**: The `/funds/aggregate` and `/funds/platform/:id` endpoints were overriding APY, gains, and currentValue from `computeFundFinalMetrics()` but not `fundSize`, causing inconsistency with the dashboard cache and platform metrics which correctly used the computed value.
