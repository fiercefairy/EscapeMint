# Release v0.41.x

### Added

- **Production mode**: `npm start` now builds all packages and serves the React UI directly from the Express API server on port 5551 — no Vite dev server needed in production
- **SPA fallback**: Non-API GET requests return `index.html` in production, enabling client-side routing on refresh

### Changed

- **`npm start` builds before serving**: Runs full build (engine, storage, server, web), flushes PM2 logs, deletes stale PM2 processes, then starts only `escapemint-api` in production mode and opens the browser
- **CDP browser is on-demand only**: Removed `escapemint-browser` from PM2 auto-start; Chrome CDP is launched on-demand via `POST /api/v1/import/browser/launch` when scraping features are used

### Removed

- **`escapemint-browser` PM2 app**: No longer auto-launched on dev or production startup
