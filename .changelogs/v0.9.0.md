# Release v0.9.0 - M1 Margin Borrowing with Auto-Sync

Released: 2026-01-11

## Overview

This release introduces intelligent margin borrowing for M1 Finance accounts with automatic synchronization, preventing cash balances from going negative while accurately tracking margin utilization and costs. The system automatically borrows from available margin when cash is insufficient, with full visibility into margin availability and borrowing.

## 🎉 New Features

### M1 Margin Borrowing System
- **Automatic Margin Borrowing**: Prevents cash from going negative by automatically borrowing from available margin when insufficient cash is available for trades
- **Two-Phase Calculation**:
  - **Initial Phase**: Handles per-trade shortfalls based on previous day's balance
  - **Auto-Sync Phase**: Handles same-day aggregate cash flow shortfalls
- **Margin Availability Tracking**: Real-time visibility into available margin across all entries
- **Platform-Specific Messaging**: Clear UI messaging for M1 vs other platforms about margin borrowing behavior

### Enhanced UI for Margin Management
- **Margin Input Fields**: Added to both "Take Action" and "Edit Action" forms
  - Displays current margin available
  - Allows manual override of auto-calculated margin borrowed
  - Shows when margin borrowing will occur automatically
- **Smart Recommendations**: Recommendation engine now treats margin as buying power
- **Validation**: Prevents margin borrowing beyond available margin limits

## 📋 Technical Details

### Margin Calculation Flow
1. **Previous Day Balance**: Calculate cash available from previous day's closing balance
2. **Initial Shortfall**: If entry amount exceeds previous day balance, borrow margin for the difference
3. **Same-Day Aggregation**: After processing all entries, check if aggregate same-day cash flow creates additional shortfall
4. **Auto-Sync Phase**: If aggregate shortfall exists, borrow additional margin proportionally across today's entries
5. **Availability Caps**: All margin borrowing is capped to actual margin available

### Margin Fields
- `margin_available`: Available margin credit line
- `margin_borrowed`: Total margin currently borrowed
- `marginBorrowedNow`: Margin borrowed for this specific entry

## 🚀 Upgrading

If you have existing cash funds with margin enabled, no data migration is required. The new margin tracking will start with your next entry.

To enable margin tracking on an existing cash fund:
1. Navigate to the fund detail page
2. Click "Edit Fund"
3. Check "Margin Enabled"
4. Set your margin APR (e.g., 0.0725 for 7.25%)
5. New entries will now track margin borrowing automatically
