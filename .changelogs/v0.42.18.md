# Release v0.42.18

Released: 2026-03-01

## Overview

Fixes inflated aggregate projected annual return by using portfolio-level time-weighted realized APY instead of summing individually-compounded per-fund projections.

## Bug Fixes

- **Aggregate projected annual return uses portfolio-level APY**: Previously calculated as `SUM(fund.currentValue * fund.realizedAPY)` where each per-fund APY was independently compound-annualized. Short-duration funds with modest returns got exponentially inflated APYs (e.g., 1% over 10 days → 43.9% annualized), producing an overstated aggregate projection. Now uses `totalActiveValue * portfolioRealizedAPY` matching the time-weighted approach already used for aggregate realized APY.
- **Dashboard cache projected return uses same fix**: The cached dashboard path (`dashboard-cache.ts`) now uses the same portfolio-level formula for consistency.
