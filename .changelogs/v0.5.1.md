# Release v0.5.1 - UI Cleanup

Released: 2026-01-06

## Overview

A minor release that removes Cash APY settings from the UI, simplifying the interface by relying on direct entry-based interest tracking.

## 🔧 Improvements

### Simplified Cash Interest Tracking
- Removed Cash APY configuration from UI settings
- Interest is now tracked directly via fund entries instead of calculated from APY rate
- Cleaner, more transparent approach to recording cash interest
- Reduces configuration complexity for cash funds

## 📦 Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```
