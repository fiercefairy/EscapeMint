# Release v0.42.20

Released: 2026-03-01

## Overview

Hardens dependency management to prevent version mismatch bugs and ensure reproducible installs for contributors.

## Changes

- **Group Dependabot updates for coupled packages**: React ecosystem (`react`, `react-dom`, `@types/react`, `@types/react-dom`) and Vitest ecosystem (`vitest`, `@vitest/*`) are now grouped so they bump together in a single PR, preventing version mismatches.
- **Add Dependabot coverage for `pages/`**: The `pages/` directory has its own `package-lock.json` outside the workspace and was not being updated by Dependabot.
- **Add peer dependency check to CI**: `npm ls` is checked for invalid peer dependencies before build/test, catching version mismatches before merge.
- **Use `npm ci` in Pages workflow**: Changed from `npm install` to `npm ci` for reproducible builds from lockfile.
- **Add `.nvmrc`**: Pinned to Node 20 (matching CI and `engines` constraint) so contributors can `nvm use` to match.

## Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```
