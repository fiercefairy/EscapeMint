# Release v0.3.0 - Platform Dashboards & CSV Import

Released: 2025-11-01

## Overview

This release adds platform-specific dashboards with aggregate metrics, CSV import functionality for Robinhood, and platform-level configuration management.

## 🎉 New Features

### Platform Dashboards
- **Platform-Specific Views**: Dedicated dashboard page for each platform
- **Aggregate Metrics**: Roll-up metrics showing total value, gains, and performance per platform
- **APY History Tracking**: Track and visualize annual percentage yield over time
- **Multi-Platform Support**: Manage investments across Robinhood, M1, Coinbase, and more

### Data Import
- **Robinhood CSV Import**: Import transaction history directly from Robinhood CSV exports
- **Automatic Parsing**: Smart detection of buys, sells, dividends, and other transactions
- **Bulk Entry Creation**: Efficiently import years of transaction history

### Platform Configuration
- **Platform-Level Configs**: Shared settings across all funds on a platform
- **Customizable Defaults**: Set default DCA amounts, target APY, and other parameters
- **Better Organization**: Group related funds under their respective platforms

## 🔧 Improvements

- Enhanced navigation with platform-based organization
- Better visibility into per-platform performance
- Foundation for platform-specific features and integrations

## 📦 Installation

```bash
git clone https://github.com/atomantic/EscapeMint.git
cd EscapeMint
npm run setup
npm run dev
```
